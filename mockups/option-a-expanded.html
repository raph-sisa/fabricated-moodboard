What is Arduino?

Module 02 · Lesson 01 · 8 min read

Arduino is an open-source electronics platform based on easy-to-use hardware and software. At its heart is a small computer called a microcontroller — a chip that can read inputs (a sensor detecting light, a finger pressing a button) and turn them into outputs (turning on an LED, spinning a motor, publishing data online).

Unlike your laptop or phone, a microcontroller runs one program at a time, in a loop, forever. This makes it perfect for dedicated tasks — monitoring temperature, controlling lights, reading sensors, driving motors. It does one thing, and it does it reliably.

// Your first Arduino sketch
void setup() {
  pinMode(LED_BUILTIN, OUTPUT);
}

void loop() {
  digitalWrite(LED_BUILTIN, HIGH);
  delay(1000);
  digitalWrite(LED_BUILTIN, LOW);
  delay(1000);
}

This is the "Hello, World" of hardware — the Blink sketch. It turns the built-in LED on for one second, off for one second, forever. Simple, but it proves something powerful: you just programmed a physical object.
