Workshop Flow
  • Intro + safety briefing (15 min)

  • Guided build with Claude Code (2.5 hrs)

Prerequisites
Who It's For
No hardware experience needed. If you can follow a recipe, you can solder a circuit. Claude Code walks you through every step — from first wire to working firmware.
