Module 02 — Arduino — Lesson 01: What is Arduino?
Arduino is an open-source electronics platform based on
easy-to-use hardware and software.
At its heart is a small computer called a microcontroller — a
chip that can read inputs (a sensor detecting light, a finger
pressing a button) and turn them into outputs (lighting an LED,
spinning a motor, publishing data online).
Unlike your laptop, a microcontroller runs one program
at a time, in a loop, forever. This makes it perfect for
dedicated tasks.
// Your first Arduino sketch — "Blink"
void setup() {
pinMode(LED_BUILTIN, OUTPUT);
}
void loop() {
digitalWrite(LED_BUILTIN, HIGH); // LED on
delay(1000); // wait 1s
digitalWrite(LED_BUILTIN, LOW); // LED off
delay(1000); // wait 1s
}
